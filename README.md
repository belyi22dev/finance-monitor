# ФинМонитор 💰

Веб-приложение для управления личными финансами: отслеживание займов, кредитов, МФО и банковских продуктов.

## 🚀 Возможности

### Основной функционал
- **Управление договорами**: Добавление, редактирование и удаление записей о кредитах и займах
- **Дашборд**: Визуализация статистики с графиками (Chart.js)
- **Поиск и фильтрация**: Живой поиск по названию организации
- **Экспорт данных**: Выгрузка таблицы в CSV формат
- **Сохранение данных**: Автоматическое сохранение в LocalStorage браузера
- **Адаптивный дизайн**: Корректное отображение на ПК и мобильных устройствах
- **Тёмная тема**: Современный интерфейс в тёмных тонах

### Уведомления и валидация
- Всплывающие уведомления (Toast) об успешных операциях и ошибках
- Полная валидация форм ввода с подсказками
- Подтверждение перед удалением записи

## 🛠 Технологии

- **HTML5/CSS3** - структура и стили
- **Bootstrap 5** - адаптивная вёрстка и компоненты
- **JavaScript (ES6+)** - логика приложения
- **Chart.js** - построение графиков
- **Google Fonts** - шрифты
- **LocalStorage** - хранение данных на стороне клиента
- **FontAwesome** - иконки

## 📦 Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/belyi22dev/finance-monitor.git
cd finance-monitor
```

2. Откройте файл `index.html` в любом современном браузере:
```bash
# Просто откройте файл двойным кликом или через Live Server
```

> ⚡ **Примечание**: Приложение работает полностью на стороне клиента, сервер не требуется.

## 📖 Как использовать

### Добавление договора
1. Нажмите кнопку **"Добавить договор"**
2. Заполните форму:
   - Название организации (обязательно)
   - Тип договора (Кредит/Займ/МФО/Банк)
   - Сумма долга
   - Процентная ставка
   - Дата выдачи
   - Срок (месяцев)
   - Статус (Активен/Закрыт)
3. Нажмите **"Сохранить"**

### Редактирование
- Нажмите на иконку ✏️ в строке таблицы для изменения данных

### Удаление
- Нажмите на иконку 🗑️ и подтвердите действие

### Экспорт данных
- Нажмите кнопку **"Экспорт в CSV"** для скачивания таблицы

### Поиск
- Введите название организации в поле поиска для фильтрации таблицы

## 🔒 Безопасность данных

Все данные хранятся локально в браузере пользователя (LocalStorage). При очистке кэша браузера данные будут удалены. Для резервного копирования используйте функцию экспорта в CSV.
